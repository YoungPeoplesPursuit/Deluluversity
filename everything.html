Stress-Free SET

I love playing SET so I made a SET clone during the summer before my senior year. I came up with the backend logic and did a lot of Googling to figure out how Pygame worked. The game loop was super confusing but I'm very proud that for the most part it's playable. This was one of those things that started off as a whim and "what if I could put OOP to work?" and ended as a supplemental.
